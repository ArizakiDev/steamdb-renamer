Apprenez à obtenir votre clé API Steam en quelques étapes simples Learn how to get your Steam API key in a few simple steps

Prérequis Prerequisites

  • Un compte Steam actif An active Steam account
  • Au moins un achat sur le compte Steam (pour vérifier qu'il est légitime) At least one purchase on the Steam account (to verify it's legitimate)
1

Connexion au portail développeur Steam Log in to the Steam Developer Portal

Rendez-vous sur le portail développeur Steam : Go to the Steam Developer Portal:

Portail Steam API Steam API Portal
2

Accepter les conditions d'utilisation Accept the Terms of Use

Lisez attentivement les conditions d'utilisation de l'API Steam. L'utilisation abusive peut entraîner la révocation de votre clé. Read the Steam API Terms of Use carefully. Misuse may result in your key being revoked.

3

Renseigner le domaine Enter the domain

Entrez le domaine sur lequel vous utiliserez l'API : Enter the domain where you'll use the API:

  • Pour une application locale : utilisez For a local application: use localhost
  • Pour un site web : utilisez votre nom de domaine For a website: use your domain name
  • Pour une application desktop : utilisez For a desktop application: use 127.0.0.1
4

Récupérer votre clé API Get your API key

Votre clé API sera affichée. Conservez-la précieusement et ne la partagez jamais ! Your API key will be displayed. Keep it safe and never share it!

Important - Sécurité Important - Security

  • Ne partagez jamais votre clé API Never share your API key
  • Ne la commettez pas dans un dépôt Git public Don't commit it to a public Git repository
  • Utilisez des variables d'environnement pour la stocker Use environment variables to store it
  • Surveillez son utilisation régulièrement Monitor its usage regularly
